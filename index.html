Unsugirls

Le mouvement des femmes qui disent non au sucre raffine

Une communaute de femmes qui veulent prendre soin d'elles. Des challenges de 30 jours pour se liberer du sucre raffine — le sucre blanc, le sucre des patisseries, le sucre de table — qui cause fatigue, crashs d'energie, problemes de peau, prise de poids, et addiction. Ensemble, elles se transforment et se surpassent.

La Big Idea

Le probleme

Le sucre raffine est un poison silencieux. Il provoque :

  • Addiction — le sucre active les memes circuits cerebraux que les drogues dures
  • Fatigue chronique — les pics et crashs de glycemie epuisent le corps
  • Problemes de peau — acne, vieillissement premature, teint terne
  • Prise de poids — le sucre se transforme en graisse stockee
  • Inflammation — terrain pour les maladies chroniques
  • Sautes d'humeur — irritabilite, anxiete, depression

Des millions de femmes veulent arreter. Mais seules, elles n'y arrivent pas. Il manque le soutien, la communaute, la structure.

La solution : Unsugirls

Une plateforme communautaire immersive ou des centaines de femmes vivent ensemble un challenge de 30 jours sans sucre raffine. Pas un regime. Pas une app de coaching froide. Une experience sociale et digitale ou chaque femme est soutenue, motivee, et celebree par les autres.


Pendant 30 jours, elles :

  • Echangent dans des salons de discussion dedies (recettes, motivation, victoires)
  • Participent a des salons vocaux en direct
  • Sont accompagnees par un coach IA personnel disponible 24h/24
  • Check-in chaque matin et maintiennent leur streak
  • Sont jumelees avec une buddy qui les soutient
  • Partagent leurs progres et celebrent chaque victoire
Le mouvement

Creer un mouvement, pas juste une app

L'objectif n'est pas de vendre un produit digital. C'est de creer un mouvement de femmes qui refusent le sucre raffine et choisissent une vie plus saine. Des femmes qui :


  • Ont une belle peau — sans l'inflammation du sucre
  • Ont une energie stable — plus de crash a 15h
  • Dorment mieux — le sucre perturbe le sommeil
  • Sont de meilleure humeur — fini les montagnes russes emotionnelles
  • Se sentent libres — l'addiction est brisee
  • Inspirent les autres — chaque Unsugirl devient une ambassadrice

"Je suis une Unsugirl" — c'est une identite, pas juste un challenge.

Le modele

Marketing d'influence

L'acquisition se fait majoritairement via des influenceuses. Des micro-influenceuses sante, nutrition, bien-etre, fitness partagent le challenge avec leur communaute. Chaque influenceuse amene une dizaine de femmes. 100 influenceuses = 1 000 participantes.


Les influenceuses ne sont pas des prestataires — elles sont des ambassadrices du mouvement. Elles participent elles-memes au challenge, partagent leur experience en direct, et deviennent la preuve vivante que ca marche.

Notifications Push — Comment ca marche

Expo Notifications → APNs + FCM

Expo Notifications est le service de notifications push integre a Expo. Il sert de proxy intelligent : on envoie une seule requete a Expo, et Expo la route automatiquement vers le bon service selon le telephone de l'utilisatrice.


ComposantRoleCout
Expo Push ServiceAPI unique pour envoyer les notifications. On lui donne le message + le token de l'utilisatrice, il fait le reste.Gratuit
APNs (Apple Push Notification service)Service d'Apple qui delivre les notifications sur iPhone/iPad. Expo envoie a APNs automatiquement pour les utilisatrices iOS.Gratuit
FCM (Firebase Cloud Messaging)Service de Google qui delivre les notifications sur Android. Expo envoie a FCM automatiquement pour les utilisatrices Android.Gratuit

Le flux : Notre app → Expo Push Service → detecte iOS ou Android → APNs (iPhone) ou FCM (Android) → notification sur le telephone


Config requise en production :

  • Une cle APNs (generee dans le portail Apple Developer) — config one-time
  • Un projet Firebase avec FCM active (gratuit) — config one-time

Exemples de notifications prevues :

  • "Tu n'as pas fait ton check-in aujourd'hui"
  • "Ta buddy Camille t'a envoye un message"
  • "Live coaching dans 15 minutes"
  • "Bravo ! Tu es a ton jour 14 sans sucre"
  • "Nouveau message dans le salon Motivation"

Detail des etapes

Etape 1 — Decouverte

L'utilisatrice decouvre UnSugar via une micro-influenceuse (Instagram, TikTok, YouTube) ou par bouche-a-oreille. L'influenceuse partage un lien vers la landing page, eventuellement avec un code promo.

Etape 2 — Landing page + Paiement

Page de vente presentant le challenge 30 jours. Paiement 29,99 EUR via Stripe. Au checkout, elle renseigne son email et son numero de telephone.

Etape 3 — Reception du code

Immediatement apres le paiement, elle recoit son code d'acces unique par email ET par SMS (double envoi pour ne jamais perdre le code). L'email contient aussi le lien de telechargement de l'app.


Outils : Brevo (ex-Sendinblue) pour l'envoi email + SMS. Cout : ~0,04 EUR/SMS.

Etape 4 — Telechargement

Elle clique sur le lien App Store ou Play Store dans l'email/SMS et telecharge l'app UnSugar.

Etape 5 — Creation de compte + Code d'acces

Dans l'app, elle passe par 4 ecrans :

  • Ecran 1 : "Bienvenue chez les Unsugirls" + bouton Commencer
  • Ecran 2 : Saisie du prenom
  • Ecran 3 : Saisie de l'email
  • Ecran 4 : Saisie du code d'acces (ex: UNSG-7K2M-X9PL)

Le code est verifie dans Supabase. Si valide → acces autorise. Si invalide → message d'erreur + bouton "Renvoyer mon code".

Code a usage unique, expire apres 7 jours.

Etape 6 — Elle est dans l'app

Le challenge commence. Elle a acces a :

  • Les 5 salons de discussion
  • Le salon vocal "Discussion libre"
  • Le coach IA personnel
  • Son premier check-in du matin
  • Sa buddy (jumelage automatique)

Logo Unsugirls — Concept

Deux ailes d'ange formees par la lettre "L"

Le logo represente deux L symetriques qui forment des ailes d'ange. Symbolique :

  • Les ailes = la liberation, l'envol, la transformation
  • Le double L = le lien entre les femmes (elles s'envolent ensemble)
  • L'ange = la purete retrouvee, la version pure de soi sans le poison du sucre
